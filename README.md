# eartharoid.me
 
My personal website. Built with Hugo + Bulma, deployed with Vercel. 

## Contribute
To contribute, fork this repo and run:

```bash
yarn
```

or

```bash
npm install
```

and

```bash
yarn dev
```

or

```bash
npm run dev
```

## Build
To build, run:

```bash
yarn build
```

or

```bash
npm run build
```
